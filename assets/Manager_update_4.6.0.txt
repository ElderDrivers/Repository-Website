EdXposed Manager v4.6.0 Changes Log

- Module Activation Scope support
- Enable Xposed Hide(Black) by default
- UI tweak
- SandHook backend variant switch (in Settings)
- Translation update
- Bugs & crashes fix
- Remove useless permissions check
- Dark mode
- Support Android R userspace reboot
- Update resource hook flag
- Module notification switch
- List sorts & filters

**Notes**:
- The enhancement mode was temporarily removed due to a persistent bug.
- Edxposed v0.5.2.1 or higher is required and is not compatible with previous versions.
